Submission notes

The following libraries were used:
  • JQuery
  • Bootflat (An alternative 'flat' to Bootstrap).
  • Font Awesome - for our icon needs.
  • Moment.js - for our time formatting needs.
Known bugs:
  • Sometimes due to connection issues with the MySQL database (Too many connections error, probably cloud problems) - content will not load as expected on pages.
  • A discount appears as an item with negative price in the final bill, and it is possible to 'remove' it, which is a bug..
  • We might have missed some validation cases in our forms.
Interesting things to look at:
  • EzRest allows restaurant staff to handle table orders, and perform various actions in the Manager Control screen such as adding new menu items, employees, VIPs and custom permissions.
  • Every employee has a permission level assigned to him upon creation. Insufficient permissions limit (disables) some options throughout the system.
  • Main page contains clocked in employees as well as available employees. Clicking a clocked in employee will take you to the tables screen, where you can select a table and add items to it, or create a new table.
